CPT 97153 Documentation Requirements: The Complete Guide for ABA Providers

By ABA Therapy Billing Clarity  ·  April 2026  ·  7 min read

CPT 97153 — Adaptive Behavior Treatment by Protocol — is the workhorse of ABA billing. For most practices, it represents 60–80% of total claim volume. It is also the most-audited ABA billing code by every major commercial payer and Medicaid program in the country.

The good news: 97153 denials and recoupments are almost entirely preventable. They stem from documentation gaps that can be identified and corrected before claims are submitted. This guide covers exactly what payers expect to find when they audit 97153 claims.

What CPT 97153 Covers

CPT 97153 is defined as adaptive behavior treatment by protocol, administered by technician, face-to-face with one patient. Each unit represents 15 minutes of direct treatment time. Key definitional requirements:

  • Administered by technician: The treating provider must be an RBT or BCaBA — not a BCBA. BCBA direct treatment time is billed under 97155.
  • Face-to-face: The technician must be physically present with the patient. Telehealth billing rules for 97153 vary by payer and state.
  • By protocol: The treatment must follow a BCBA-designed behavioral intervention protocol that exists in the patient's treatment plan.
  • One patient: Group sessions (two or more patients simultaneously) are billed under 97154, not 97153.

The Six Documentation Elements Payers Audit

1. Start and End Times

Every 97153 session note must include the exact start and end time of the session. This is non-negotiable — without start and end times, payers cannot verify that the number of units billed corresponds to time actually delivered. A 45-minute session = 3 units. A 30-minute session = 2 units. Rounding up is a common audit finding.

Common error: Some practice management systems record only session duration, not clock times. Payers require actual start and end times — "9:00 AM to 10:15 AM," not just "75 minutes."

2. Behavioral Targets Addressed

Session notes must identify the specific behavioral targets addressed during the session. These targets must exist in the current, authorized treatment plan. Generic language such as "worked on behavioral goals" or "implemented treatment plan" is insufficient and is a common reason for post-payment recoupment demands.

Each note should reference specific target behaviors by name — the same names used in the treatment plan — and include data reflecting progress or performance on each target addressed.

3. Data Collection

ABA is a data-driven discipline, and payers expect session notes to reflect that. Acceptable data formats include discrete trial data, frequency/rate data, duration data, interval data, and percentage correct. The data format should match what is specified in the treatment plan for each target.

Session notes that describe behavior qualitatively without quantitative data are frequently flagged during audits. "Patient was cooperative and engaged" is not data. "Patient responded correctly on 8 of 10 trials for target X" is data.

4. Technician Credentials and Signature

The session note must be signed by the technician who delivered the service, with their credentials clearly identified. For RBT-delivered services, the note must include the technician's RBT number or certification status. Electronic signatures are acceptable to most payers provided they are tied to a unique provider identifier.

5. BCBA Supervision Documentation

This is the most commonly deficient element in 97153 audits. Payers require documentation that the delivering technician was under active BCBA supervision — not just that a BCBA exists somewhere in the practice. Required supervision documentation includes:

  • The supervising BCBA's name and credential number
  • Dates and duration of supervision contacts for this technician
  • Evidence that supervision meets payer-required frequency (commonly 5–10% of service hours)
  • Documentation that the BCBA reviewed and signed off on the treatment plan being implemented

6. Authorization Verification

Each 97153 claim must correspond to an active prior authorization that covers the service date, the CPT code, and the number of units billed. Billing against an expired authorization, exceeding authorized units, or billing a service date outside the authorization period are among the most common reasons for 97153 denials.

97153 Session Note Checklist

  • Exact session start and end time recorded
  • Units billed match actual time delivered (no rounding up)
  • Specific behavioral targets addressed, named as in treatment plan
  • Quantitative data for each target addressed
  • RBT/technician name, credentials, and signature
  • Supervising BCBA identified by name and credential
  • Active prior authorization covers this date and code
  • Units billed do not exceed remaining authorized units

Common 97153 Billing Errors That Trigger Audits

Unit Miscounting

The most common 97153 error. Billing 4 units for a 55-minute session (which should be 3 units), or billing 3 units for a session that was interrupted and actually only lasted 35 minutes. Payers cross-reference units billed against session notes — discrepancies trigger immediate review.

Unbundling

Billing multiple 97153 claims for a single continuous session — for example, billing two separate 97153 claims of 2 units each instead of one claim for 4 units for the same date. Payers flag this as potential upcoding even when it occurs accidentally through system errors.

Concurrent Billing

Billing 97153 and 97155 at the same time for the same patient. These codes cannot be billed concurrently. 97153 is technician time; 97155 is BCBA direct treatment time. If a BCBA steps in to deliver treatment directly, that time is 97155 — the technician's 97153 time should stop.

Protocol Mismatch

Delivering and billing treatment for behavioral targets not included in the current authorized treatment plan. If a new target is added clinically before the treatment plan is updated and authorized, claims for that target may be denied retroactively.

Payer-Specific 97153 Requirements

Beyond the universal documentation requirements above, major commercial payers impose additional requirements that vary significantly:

  • UnitedHealth: Requires session notes to be completed within 24 hours of service delivery. Backdated notes are a compliance risk.
  • Aetna: Imposes strict supervision ratio requirements and may request supervision logs as part of routine audits.
  • BCBS: Plan requirements vary significantly by state — BCBS of Illinois has different requirements than BCBS of Texas.
  • Tricare: Has specific provider type requirements and may require prior authorization for hours above a weekly threshold.
  • Medicaid: State-specific requirements vary widely — some states require electronic visit verification for all home-based 97153 sessions.
