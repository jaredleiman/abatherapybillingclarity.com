CPT Code 97153: ABA Adaptive Behavior Treatment Documentation Guide

By ABA Therapy Billing Clarity  ·  April 2026  ·  8 min read

CPT 97153 — Adaptive Behavior Treatment by Protocol — is the workhorse code of ABA billing. For most practices, it represents 60-80% of total claim volume. It is also the code that payers audit most aggressively. Understanding exactly what documentation is required for every 97153 claim is not optional — it is the foundation of your practice's financial survival.

What CPT 97153 Covers

CPT 97153 is defined as adaptive behavior treatment by protocol, administered by a technician (RBT or BCaBA), face-to-face with one patient. Each unit represents 15 minutes of direct treatment time. The key definitional requirements are:

  • Administered by technician — The treating provider must be an RBT or BCaBA, not a BCBA. BCBA direct treatment is billed under 97155.
  • Face-to-face — The technician must be physically present with the patient. Telehealth rules for 97153 vary by payer and state.
  • By protocol — Treatment must follow a BCBA-designed protocol that exists in the patient's treatment plan.
  • One patient — Group sessions are billed under 97154, not 97153.

The Six Documentation Elements Payers Audit

1. Session Start and End Times

Every 97153 session note must include exact start and end times. Units billed must match the documented time. For a 2-hour session (120 minutes), you bill 8 units. The 8-minute rule applies — a partial unit is billable only if at least 8 minutes of the 15-minute unit were provided.

2. Target Behaviors and Programs Run

The session note must identify which specific target behaviors from the treatment plan were addressed and which skill acquisition programs were run. Generic notes like "worked on communication goals" are insufficient. Specify: "Ran DTT program for manding using PECS Phase III; ran NET program for play skills — parallel play with preferred items."

3. Data Collection

Raw data must be collected during the session and referenced in the note. Payers expect to see trial-by-trial data, frequency counts, duration measures, or interval recording — whichever matches the measurement method specified in the treatment plan. Session notes that reference "data collected" without actual data are audit failures.

4. Client Response and Progress

Document how the client responded to each program. Include correct/incorrect response ratios, prompted vs. independent responses, and any notable behavioral events. This is what connects today's session to the treatment plan's goals and demonstrates that treatment is producing measurable change.

5. Protocol Modifications

If the RBT made any in-session adjustments based on BCBA-approved decision rules (e.g., moving to a new prompt level, adjusting reinforcement schedules), document what was changed and why. These modifications should align with the supervision notes from the BCBA.

6. Supervision Linkage

The session note should be linkable to a corresponding BCBA supervision entry. Auditors check whether the supervision ratio (typically 5-10% of direct hours) is being met and whether supervision activities connect to the 97153 treatment being delivered.

Audit reality: The most common 97153 denial is not a documentation error — it is a data collection gap. Payers deny claims when session notes lack objective data showing what was measured, what the baseline was, and whether the client's performance is changing over time.

Common Billing Errors

  • Billing 97153 for BCBA direct services — If a BCBA is providing direct treatment, the correct code is 97155, not 97153
  • Billing 97153 during group sessions — Two or more clients receiving services simultaneously requires 97154
  • Billing non-face-to-face time — Drive time, note writing, and preparation are not billable under 97153
  • Rounding up partial units — The 8-minute rule must be applied; you cannot round 5 minutes into a billable unit
  • Missing treatment plan linkage — Every 97153 session must connect to active goals in a current, authorized treatment plan

Payer-Specific Requirements

While the CPT definition of 97153 is universal, individual payers may layer additional requirements on top. Common payer-specific variations include requirements for parent signature on session notes, specific session note templates or formats, minimum session duration requirements (some payers won't pay for sessions under 1 hour), and different unit calculation rules for Medicaid vs. commercial claims.

Always verify your top payers' specific 97153 documentation requirements beyond the CPT standard.

The Bottom Line

CPT 97153 is simultaneously the most important and most vulnerable code in your billing. The practices that protect their 97153 revenue are the ones that treat session notes as compliance documents, collect real data at every session, and ensure every note connects clearly to an authorized treatment plan with measurable goals. The few extra minutes spent on thorough documentation pay for themselves many times over in prevented denials and successful audits.
