ABA Billing Codes: Complete CPT Code Reference for BCBA Practices

By ABA Therapy Billing Clarity · April 2026 · 10 min read

ABA therapy uses a specific set of CPT codes — the 97151-97158 series — designed exclusively for adaptive behavior services. Each code has distinct definitions, billing rules, and documentation requirements. This guide is a complete reference for every code in the series.

97151 — Behavior Identification Assessment

This code covers the initial and periodic comprehensive assessment. The BCBA conducts direct observation, administers standardized assessments (VB-MAPP, ABLLS-R, AFLS), completes the Functional Behavior Assessment, and develops the treatment plan. Typically authorized for 8-32 units per assessment period. Used at intake and at re-assessment intervals (usually every 6 months). See our detailed 97151 guide.

97152 — Behavior Identification Supporting Assessment

This is the technician-level assessment code. An RBT or BCaBA conducts structured observations, preference assessments, or other assessment activities under BCBA direction. Not all payers cover 97152 — verify before billing. When covered, it is typically used for data collection activities that support the BCBA's assessment but do not require BCBA-level clinical judgment.

97153 — Adaptive Behavior Treatment by Protocol

The primary direct therapy code. RBTs and BCaBAs deliver 1:1 treatment following the BCBA's treatment protocols. This is the highest-volume code in most practices, representing 60-80% of all claims. Requires detailed session notes with data collection, target behaviors addressed, and client response documentation. See our detailed 97153 guide.

97154 — Group Adaptive Behavior Treatment

Group therapy with 2 or more clients receiving ABA services simultaneously. Each client must have individual goals being addressed within the group context. Documentation must reflect individualized data collection for each client, even in the group setting. Not all payers cover group ABA — Medicaid coverage varies by state, and some commercial plans exclude it.

Group billing rule: When billing 97154, you bill per client, not per group. If three clients are in a 1-hour group, you bill 4 units of 97154 for each of the three clients — not 4 units total.

97155 — BCBA Direct Supervision

BCBA supervision during treatment sessions. The BCBA observes the RBT, modifies protocols in real time, models techniques, and directs treatment. Face-to-face with the client is required. Supervision ratios (typically 5-10% of direct hours) must be maintained. See our detailed 97155 guide.

97156 — Caregiver Training

BCBA-delivered training for parents, guardians, or other caregivers. Covers teaching caregivers to implement behavior strategies, use reinforcement systems, manage challenging behaviors, and support skill generalization at home. Face-to-face with the caregiver (not the client). See our detailed 97156 guide.

97157 — Multiple-Family Group Guidance

Group caregiver training with multiple families simultaneously. Rarely used in practice. When used, each family's goals should be addressed, and documentation must reflect individualized guidance within the group format. Most payers do not specifically authorize 97157 — verify coverage before billing.

97158 — Group Protocol Modification

BCBA supervision and protocol modification in a group treatment setting. Similar to 97155 but applied when the BCBA is supervising group therapy (97154) rather than individual therapy (97153). Used infrequently and not covered by all payers.

Codes That Cannot Be Billed Together

Several code combinations create billing conflicts. The same provider cannot bill 97153 and 97155 simultaneously — an RBT bills 97153 while the BCBA bills 97155 during the same session, but one person cannot bill both. 97155 and 97156 cannot overlap — the BCBA is either supervising the session (97155) or training the caregiver (97156), not both at the same time. 97151 and 97155 same-day restrictions apply with some payers.

The Bottom Line

Knowing the correct code for each activity is the foundation of ABA billing. The most common errors come from using the wrong provider level (billing BCBA time under an RBT code), confusing face-to-face requirements (97155 requires client presence, 97156 requires caregiver presence), and billing codes that the specific payer doesn't cover. When in doubt, check the payer's ABA policy manual — they all have one, and the specifics matter.
